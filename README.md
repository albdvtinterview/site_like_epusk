# ЭНЕРГОПУСК — фронтенд-витрина

Одностраничная адаптивная витрина промышленного оборудования на React, Vite, Zustand и Tailwind CSS. API, база данных, личный кабинет и корзина не используются.

## Запуск

```bash
npm install
npm run dev
```

## Production-сборка

```bash
npm run build
```

Готовые статические файлы появятся в папке `dist` и подходят для размещения на обычном статическом хостинге.

## Замена изображений

Все места под изображения отмечены атрибутом `data-image-slot` и текстовой подписью. Компонент находится в `src/components/ImagePlaceholder.jsx`. Замените его вывод на `<img>` или используйте плейсхолдер точечно в нужных карточках.

## Контактная форма

Форма не отправляет данные на сервер. Она формирует `mailto:`-ссылку и открывает почтовое приложение пользователя с заполненным письмом.
